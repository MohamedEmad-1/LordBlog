Evelx Logs Of A Random Dev V1 was the first public release of this theme, focused on a clean visual identity and better post navigation.

## Intro Animation

The first version introduced an animated hero/intro treatment to make the opening state feel more intentional.

Key details:

- Entry animation plays immediately when the page loads.
- Motion is tuned to stay lightweight so it does not block reading.
- The animation is decorative only; content remains accessible while motion runs.

## Tag System Changes

V1 formalized tags as a first-class part of content discovery.

Changes included:

- Standardized `tags` frontmatter usage across posts.
- Improved consistency of tag labels for filtering and archives.
- Better support for tag-driven navigation so users can explore related topics quickly.

## Table of Contents Always Showing

In this release, the table of contents is presented as always visible content, not hidden behind interaction.

This improves:

- Scanability for long posts.
- Fast jumping between sections.
- Reader orientation while scrolling.

## Other Foundations Added in V1

- Minimal and fast blog structure
- Markdown-first publishing flow
- Light and dark theme support

Evelx Logs Of A Random Dev V1 established the base design language and release structure that later versions continued to improve.
